# 🪨📄✂️ Rock Paper Scissors Game

A simple interactive Rock-Paper-Scissors game built using **HTML**, **CSS**, and **JavaScript**. This game allows a player to compete against the computer in real-time with a user-friendly interface and score tracking.

## 🚀 Features

- Play against a computer opponent.
- Dynamic emoji-based buttons for Rock (👊), Paper (🖐️), and Scissors (✌️).
- Real-time result and score updates.
- Styled for a clean and responsive layout.
- Visual win/lose feedback with color indicators.

## 🛠️ How to Run

1. **Clone or download** this repository.
2. Open `index.html` in any modern web browser.
3. Start clicking the buttons to play!

---

## 🧠 Game Logic Summary

- The player selects rock/paper/scissors.
- The computer randomly selects a choice.
- The result is calculated and displayed:
  - "YOU WIN!", "YOU LOSE!", or "IT'S A TIE!"
- Scores are updated accordingly.
- Color indicators:
  - ✅ Green for wins
  - ❌ Red for losses
